# FairPrice-DLE

A personal web application inspired by the popular online game [**COSTCO-DLE**](https://costcodle.com/), adapted for a Singaporean context using real grocery data from FairPrice.

This project was motivated by the lack of familiarity with COSTCO products, which made the original game less engaging. To address this, a local version was developed to allow users to guess the prices of everyday items from one of Singapore's most popular supermarkets, FairPrice.

[<u>Game is available here</u>](https://fairprice-dle.netlify.app/)

---

## Features

- **Interactive Game Interface (React + TypeScript)**

  - Clean and responsive UI for guessing item prices.
  - Feedback on guesses (e.g. higher / lower hints).

- **Web Scraping (Selenium)**

  - A scraper to extract product names and prices from FairPrice's website.
  - Enables the game to stay relevant with real-world pricing data.

---

## Tech Stack

- **Frontend:** React, TypeScript  
- **Web Scraping:** Selenium (Python)  
- **Deployment:** Netlify
